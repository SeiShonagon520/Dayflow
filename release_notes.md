## Dayflow v1.5.1

### 📊 统计页面全新设计

- 全新仪表盘风格，精致质感，告别 AI 廉价感
- 顶部指标卡片：总时长、平均效率、深度工作次数、活动数量（支持周对比）
- 双栏网格布局：左侧图表区 + 右侧数据区
- 环形图：类别时间分布可视化，支持悬停交互
- 折线图升级：渐变填充 + 贝塞尔曲线平滑
- 柱状图升级：渐变色彩 + 背景轨道
- 热力图精简：更紧凑的时段效率展示

### ✏️ 卡片编辑功能

- 支持右键菜单编辑/删除活动卡片
- 可修改：类别、标题、摘要、效率评分
- 点击卡片直接进入编辑模式

### 🔧 其他改进

- 周对比功能：本周 vs 上周数据对比
- 卡片阴影效果：更有层次感的视觉体验
- 代码优化和 Bug 修复

### 📦 安装说明

1. 下载 `Dayflow_v1.5.1.zip`
2. 解压到任意目录
3. 运行 `Dayflow/Dayflow.exe`

### 💡 API 配置

支持任意 OpenAI 兼容接口：
- OpenAI: `https://api.openai.com/v1`
- DeepSeek: `https://api.deepseek.com/v1`
- 心流 API: `https://apis.iflow.cn/v1`
- Ollama: `http://localhost:11434/v1`
